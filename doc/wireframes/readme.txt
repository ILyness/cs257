Description of each wireframe:
1 - Gives view of current performance list for all events, allowing for filtering to specific events.
2 - Gives data for the final performance list from previous years, allowing for filtering by year and event.
3 - Page that has options for various tools, such as converting marks across events and calculating decathlon scores.
4 - Has search bar on top to access tools, performance lists, or search for athletes. Also has performance list on home page.
5 - Advanced search page that allows you to look for performances by athlete, event, school, and mark.
6 - After searching for athletes/performances, there is an option to add another page for easy direct comparison.
